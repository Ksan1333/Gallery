# Gallery アプリ概要

対象バージョン: 0.6.0

## Galleryとは

Galleryは、端末内の画像、GIF、動画、本、イラスト制作の参考資料をまとめて扱うAndroidアプリです。大量のメディアを素早く見渡し、AI分析やフォルダ整理を使って必要なものへ戻りやすくすることを目的としています。

## 主な機能

### メディア一覧

- 画像、GIF、動画を一つのタイムラインに表示
- 2 / 3 / 4 / 7 / 28列の表示切り替え
- 2列では画像比率を保った余白の少ない表示
- 日付、名前、サイズによる並び替え
- メディア種別、年齢区分、縦横比、タグによる絞り込み
- スクロールバーの直接ドラッグと年月日プレビュー
- 長押しと画面端スクロールによる範囲選択

### 類似画像グループ

- AI特徴ベクトルが60%以上一致する、日時順で隣接した画像を連鎖的にまとめる
- グループ件数に上限を設けない
- 古い画像から新しい画像の順で保持
- グループタップで4列・最大2段の吹き出しをギャラリー表示順で表示
- 吹き出しから開いたビューアはグループ内画像だけを同じ順番で表示
- 9件目以降は吹き出し内スクロール
- 全体設定から表示ON/OFFを切り替え

### フォルダ

- 保存先別にメディアを表示
- DCIM直下へのフォルダ作成
- フォルダ表示順と代表サムネイルの管理
- 複数フォルダを名前付きグループへまとめる
- グループタップで構成フォルダを吹き出し表示
- グループ解除、メディアの一括移動

### ビューア

- 画像、GIF、動画の全画面表示
- ズーム、回転、スライドショー、コマ送り、フレーム送り
- 関連画像と閲覧履歴
- ビューアから戻った位置の復元
- ブックビューアのしおり、ページ移動、表示方向、平滑化

### AIとMy List

- 通常AI / 高性能AIの選択
- タグ、年齢区分、特徴ベクトルの生成
- 通常AIで解析した画像の高性能AIによる再分析
- お気に入り、未整理、未分析、タグ別カテゴリ
- 分析進捗とキャンセル

### X / Twitter保存

- 共有、リンク起動、貼り付け、手入力に対応
- MP4またはGIFとして保存
- 投稿内に動画・GIFが複数ある場合は全件保存
- 同じ動画の画質違いは1件にまとめ、選択品質または最高品質を採用
- GIF由来MP4だけを必要に応じて高品質GIFへ変換
- 保存履歴とアプリ内プレビュー

### イラスト参考資料

- 描画・イラスト制作で一時的に集めたい参考資料をプロジェクト単位で整理
- Web検索または端末ギャラリーから資料を追加
- 参考資料だけをまとめたビューアで確認

### お気に入り

- お気に入りクリエイターとお気に入りサイトを管理
- URL取得操作を明示的に実行
- 設定読み込み時は既存データを削除せず、重複を避けて追加

### 設定とテーマ

- システム、ライト、ダークテーマ
- プリセットと個別カラー編集
- 背景と文字が同色になった場合の自動コントラスト補正
- Materialメニュー、ダイアログ、ボタンを同じパレットへ統一
- ビューア操作、タッチ領域、ボトムバー、長押し時間の設定
- 設定・お気に入り・表示グループのバックアップ

## 画面構成

| 画面 | 役割 |
| --- | --- |
| ホーム | 全メディアの一覧、検索、類似グループ |
| フォルダ | 保存先別一覧、フォルダグループ、メディア移動 |
| 動画 | 動画専用一覧、プレビュー、全画面再生 |
| My List | お気に入り、未整理、AI状態、タグ分類 |
| 本 | ZIP/PDF本棚とブックビューア |
| 参考資料 | イラスト用参考資料プロジェクト |
| ゴミ箱 | 復元と完全削除 |
| Video Downloader | X投稿の動画・GIF保存と履歴 |
| 設定 | 表示、操作、AI、テーマ、バックアップ |
