# Gallery アプリ概要

対象バージョン: 1.4.0

Gallery は、端末内の画像、GIF、動画、ZIP/PDF の本、制作資料を扱う Android アプリです。大量のメディアを一覧し、閲覧、整理、AI 分析、X 投稿からのダウンロードを行えます。

## 主な機能

### メディア一覧

- 画像、GIF、動画を 2 / 3 / 4 / 7 / 28 列で表示
- 日付、名前、サイズによる並び替えと、メディア種別・年齢区分・タグなどの絞り込み
- 類似画像グループ、長押し選択、範囲選択、一括編集
- スクロールバーのドラッグ中に日付を表示し、最下端はグリッドの実下端まで一致させる

### ビューアと関連メディア

- 画像／GIF のズーム、動画再生、フレーム保存、壁紙、タグ、お気に入り、削除
- 視覚類似候補とランダム候補をビューア内パネルに表示
- GIF 変換は動画用の操作であり、3 点メニューから実行する。静止画と GIF では非活性表示

### AI 分析

- AI タグ、年齢区分、特徴ベクトルを生成
- 期間を指定してバックグラウンド分析を開始し、通知と進捗オーバーレイから状態を確認・キャンセル

### その他

- フォルダ、ゴミ箱、フォルダ移動、フォルダグループ
- X URL から画像・動画・GIF を保存。動画投稿では投稿サムネイルを保存候補から除外
- ZIP/PDF 本、制作資料プロジェクト、作家・サイト、設定、JSON バックアップ、更新確認

## 画面構成

| 画面 | 役割 |
| --- | --- |
| ホーム／フォルダ／動画 | メディア一覧と検索・絞り込み |
| 本 | ZIP/PDF 本棚とビューア |
| ゴミ箱 | 復元と完全削除 |
| 参考資料 | 制作資料プロジェクトの作成・収集・閲覧 |
| Video Downloader | X URL の候補解決、保存、履歴 |
| 設定／アプリ情報 | 表示・操作・テーマ・バックアップ・更新 |
