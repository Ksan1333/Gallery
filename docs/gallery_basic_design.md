# Gallery 基本設計

## 2026-06-30 追記: 動画プレビュー/動画ビュワー

### 目的

動画専用画面では、フォルダ内の動画を素早く確認し、必要に応じて専用の全画面ビュワーで再生、シーク、コマ送り、スクリーンショット保存を行えるようにする。

### 動画プレビュー

- フォルダ内グリッドで選択した動画を `VideoMiniPlayer` でプレビュー再生する。
- プレビュー右上に閉じるボタン、左上に全画面表示ボタンを配置する。
- 動画枠の下端に、動画枠と一体化した細いシークバーを重ねて表示する。
- プレビュー動画が終端に到達した場合は、自動的にプレビューを閉じる。

### 動画全画面ビュワー

- 全画面表示は `VideoFullscreenViewerScreen` による動画専用ビュワーとして扱う。
- 下部に再生/停止、前の動画、次の動画を配置し、左側にスクリーンショット、右側に画面回転を配置する。
- ボタンごとの個別円背景は使わず、動画上に直接アイコンを重ねる。
- 右上の一覧アイコンから、現在フォルダ内の動画サムネイル一覧を表示する。
- 縦画面では上部に横スクロール、横画面では右端に縦スクロールで動画一覧を表示する。

### ジェスチャ

- 左右スワイプは前後動画移動ではなく、1 フレーム単位のコマ送りに使う。
- コマ送り中は再生を一時停止し、指を離した時点で元が再生中だった場合は再開する。
- 右端の上下スワイプは音量調整、左端の上下スワイプは明るさ調整に割り当てる。
- 音量または明るさを調整している間は、中央オーバーレイに種類とパーセンテージを表示する。

### 表示・状態

- 全画面中は Android のシステムバーを非表示にし、終了時に復帰する。
- 操作がない状態が数秒続いた場合はコントロールを自動的に非表示にする。
- 再生位置表示はミリ秒まで表示し、コマ送り時の変化を確認しやすくする。
