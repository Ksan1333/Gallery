# 共通UI仕様

## 共通部品

- シークバーは動画ビュワーのシークバーを共通部品化し、メディア・動画・本の各ビュワーで再利用する。
- トップバーは対象画面で共通部品化し、左端のサイドバー表示可否、左揃えタイトル、右端アイコンと機能を画面ごとに差し替える。

## 画面別ボタン配置

### ホームギャラリー

- トップバー右端: 分析、並び替え、検索

### メディアビュワー

- ボトム右端: ごみ箱、設定、回転、お気に入り、3点ボタン
- 3点ボタン: スライドショー、GIF、ascii2d、スクリーンショット、壁紙、フォルダのサムネイル、タグ

### フォルダ画面

- トップバー右端: フォルダ作成

### フォルダ内ギャラリー

- トップバー右端: 並び替え

### 動画画面

- トップバー右端: 設定
- 設定は動画設定へ直接遷移する。

### 動画フォルダ内ギャラリー

- トップバー右端: 並び替え、設定

### 動画プレビュー画面

- 左上: 全画面
- 右上: 閉じる

### 動画ビュワー

- 左上: 閉じる
- 中央: ファイル名
- 右上: なし
- 画面下部: シークバー
- シークバー上: 再生位置 / 総時間
- シークバー下: カスタマイズ可能な 5 つのボタン（初期設定：回転、前の動画、再生/停止、次の動画、3点ボタン）
- 3点ボタン（メニュー内）: 閉じる、設定、スクリーンショット、動画ストリップ（一覧）表示切替

### 本画面

- トップバー右端: しおり、並び替え、設定

### 本画面フォルダ内ギャラリー

- トップバー右端: しおり、並び替え、設定

### 本ビュワー

- トップバー左端: 閉じる
- トップバー中央/左寄せ: タイトル
- トップバー右上: 3点ボタン
- 3点ボタン: お気に入り、設定、前の本、次の本、しおり
- 画面下部: シークバー
- シークバー左: 現在のページ/全体ページ
- シークバー下の左: 前の本
- シークバー下の右: 次の本

### お気に入りクリエイター・サイト

- トップバー右端: ファイル書き込み、ファイル読み込み、編集

### 設定

- トップバー右端: ファイル書き込み、ファイル読み込み、リセット

## サイドバー表示対象

以下の画面では、トップバー左端にサイドバーアイコンを表示し、右スワイプでサイドバーを開ける。

- ホーム
- フォルダ
- 動画
- 本
- ゴミ箱
- お絵描き資料
- お気に入りクリエイター
- お気に入りサイト
- 本のしおり
- 動画DL
- 設定

フォルダ内ギャラリー、動画フォルダ内ギャラリー、本フォルダ内ギャラリーではサイドバーを表示しない。上記以外の画面でもサイドバーは開けない。

## アプリ内ナビゲーションバー表示対象

アプリ内ナビゲーションバーは全体設定からカスタマイズ可能です。デフォルトでは以下の画面で表示されますが、設定で変更した場合は割り当てられたすべての主要画面で表示されます。

- ホーム
- フォルダ
- 動画
- 本
- ゴミ箱
- その他設定で割り当てられた画面（お絵描き資料、しおり、設定など）

フォルダ内ギャラリー、動画フォルダ内ギャラリー、本フォルダ内ギャラリーでは表示しない。上記以外の画面でもアプリのナビゲーションバーアイコンは表示しない。

## OSシステムバー・OSナビゲーションバー

以下のビュワー画面では、コントロールパネル表示中、または設定で表示に変更された場合のみ、OSのシステムバーとOSのナビゲーションバーを表示する。

- メディアビュワー
- 動画ビュワー
- 本ビュワー

その他の画面では、OSのシステムバーとOSのナビゲーションバーを常に表示する。
