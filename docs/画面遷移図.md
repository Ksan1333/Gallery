# Gallery 画面遷移図

## 1. 目的

Gallery の画面遷移は機能ごとに分けて確認する。全体を1枚にまとめると情報量が多くなるため、各詳細設計書の中に「機能内画面遷移図」として、画面タイトル付きのミニUIモック、Dialog、Popup、BottomSheet、遷移矢印を配置する。

このページは、各機能の遷移図へ移動するための索引である。

## 2. 表現ルール

| 表現 | 意味 |
| --- | --- |
| スマホ型のミニUI | 画面または画面内ビュー |
| 点線枠カード | `AlertDialog` / `Dialog` |
| 黄系カード | `DropdownMenu` / `Popup` |
| 緑系カード | `ModalBottomSheet` / パネル / 完了復帰 |
| 矢印 | ユーザー操作、`navigate()`、画面内状態変更 |
