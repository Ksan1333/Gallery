# 更新履歴 (CHANGELOG)

## [0.8.0] - 2026-07-20

### 動画ビューア
- シークプレビュー機能を大幅に強化しました。
  - **ハイブリッド・キャッシュ**: シークバー操作時は動画全体をカバーする「25分割・高速キャッシュ」、画面左右スワイプ時は現在地周辺を精細に映す「10分割・高画質キャッシュ（720p）」を自動で使い分けるようにしました。
  - **ローリング・キャッシュ**: 画面スワイプを開始した瞬間、前後30秒（合計1分間）の範囲を高画質で即座に抽出し、遅延のない鮮明なフレーム移動を可能にしました。
- 動画フォルダ画面のトップバーをメイン画面と共通のデザインに統一し、設定アイコンやタイトル表示の整合性を改善しました。

### ブックビューア
- Kindleスタイルの高度なシークナビゲーションを実装しました。
  - **静止メイン画面**: シークバー操作中、背後のページを固定したまま、中央のプレビューカードで内容を確認できるようにしました。現在地を見失うことなく高速にページを探せます。
  - **全ページ・プリキャッシュ**: 本を開くと同時に全ページの軽量サムネイルを裏側で生成し、シーク中の読み込み待ちを解消しました。
  - **シークアンカー（記憶点）**: シークを開始した地点に「アンカー」を自動配置し、磁石（スナップ）効果で正確に戻れるようにしました。スナップ範囲を1%に微調整し、1ページ単位の移動と吸着のしやすさを両立しました。
- 設定画面にアンカーの有効化スイッチと、最大記憶数（1〜5個）のカスタマイズ項目を追加しました。

### 設定・UI
- 設定画面の構成を再編し、ナビゲーション順序を「全般 -> テーマ -> メディア -> 動画 -> ブック」に整理しました。
- 設定画面の各ラベルに `galleryTypography` を適用し、視認性を高めるためフォントサイズを拡大しました。
- キャッシュ管理を独立したセクションに分離し、「キャッシュの消去」機能の動作（サムネイルやバッファの削除、データベースの保持）に関する説明を詳細化しました。

## [0.7.0] - 2026-07-12

### ギャラリー・フォルダ操作
- フォルダ画面のデフォルト表示を3列に変更しました。
- フォルダを長押しドラッグして自由に並び替えられるようにしました。ドラッグ直後に位置が戻ってしまう不具合も修正しました。
- フォルダ内のメディアをすべて移動または削除して空になった際、自動的にフォルダ一覧画面へ戻るようにしました。
- フォルダ移動時の移動先選択画面でも、フォルダグループや並び替え順序が正しく反映されるように改善しました。
- フォルダ画面で長押しメニューによる操作と、ドラッグによる並び替えを両立できるよう調整しました。競合していた3点ボタンは削除し、UIをクリーンに保ちました。
- フォルダのグループ化機能を改善し、既存のグループに新しいフォルダを直接追加できるようになりました（1つのグループと他の項目を選択してグループ化した場合）。
- グループ内のタイトル横の編集アイコンから、グループ名を変更できるようになりました。
- グループ解除およびグループ内フォルダの個別解除時に確認ダイアログを表示するようにし、誤操作を防止しました。
- グループ操作（作成・追加・解除・名前変更）および並び替えの結果が画面に即座に反映されるように改善しました。
- 類似画像グループ化の判定基準（一致率）を設定画面から 1%〜100% の範囲で調整できるようにしました。

## [0.6.0] - 2026-07-12

### 2026-07-12 追記

#### ギャラリー・ファイル操作
- フォルダ移動先をフォルダ名だけで `DCIM` 配下へ決めず、既存メディアの実際の `RELATIVE_PATH` から解決するようにしました。移動後の保存先検証、Android 11以降の書き込み同意、失敗件数表示、`FOLDER_MOVE_TRACE` ログも追加しました。
- AIベクトルが60%以上一致する日時順の隣接画像を、件数上限なしで連鎖的に類似画像グループへまとめる表示を追加しました。設定からグループ化の有効・無効を切り替えられます。グループタイルと吹き出し内は現在のギャラリー順で並べ、吹き出しから開いたビューアはそのグループ内だけを同じ順番で表示します。吹き出し表示時のintrinsic計測クラッシュも修正しました。
- 複数フォルダを表示上のグループへまとめ、グループタイルから吹き出し式の一覧を開けるようにしました。実フォルダや保存中のメディアは移動せず、グループ解除後も元のフォルダ構成を保ちます。
- 2列表示を段違いグリッドへ変更し、横並び画像の高さを揃えず、各画像の縦横比を保ちながら空いた領域を次の画像で埋めるようにしました。
- 4列から7列、7列から28列へ切り替える際は、負荷の高いアイテム移動・サイズ変更アニメーションを行わないようにしました。
- スクロールバーは短いタップでは移動せず、長押し後だけドラッグへ切り替わるようにしました。長押し判定と追従間隔を調整し、`GALLERY_SCROLLBAR_TRACE` で操作を追跡できます。
- ギャラリーのサムネイルデコードサイズ、近傍先読み、スクロール中のロード抑制を見直し、表示開始と高速スクロールの応答性を改善しました。
- 検索画面に過去の検索条件を復元できる履歴を追加し、保存件数を設定から1〜10件で変更できるようにしました。

#### メディア・動画・ブックビューア
- メディアビューワーでレコメンドを表示したままでも、画像領域の左右スワイプで前後のメディアへ移動できるようにしました。ビューワー表示中はサイドバーを開かないようにしました。
- 動画ビューワーの再生ボタン左右に、1フレーム単位であることが分かる前後フレーム送りボタンを追加しました。
- 動画ビューワーの3点メニューは操作後に自動で閉じないようにし、トップバーの設定ボタンとコマ送りボタンを整理しました。
- ブックビューワーの全分割タイプにタッチ様式の初期割り当てを用意し、しおり追加・切り替え・解除とトップバーからの回転・スライドショー操作を整理しました。
- 画像平滑化設定の各方式について、処理内容と向いている画像を箇条書きで確認できる説明を追加しました。

#### AI分析・ダウンロード・表示
- AI分析実行ダイアログから普通AIと高性能AIを選択できるようにし、普通AIで解析済みの画像を高性能AIで再解析できるようにしました。
- XのGIF投稿はMP4保存と高品質GIF変換を選べるようにし、全体パレット生成とディザリングを使った変換へ変更しました。GIF変換の進捗も処理段階に合わせて更新します。
- Xの投稿から動画・GIFを複数検出した場合、メディアごとに画質を選び、全件を別ファイル・別履歴として保存できるようにしました。
- 円形進捗表示を移動した際にタップ領域が元位置へ残らないようにし、円の背景へ透明度20%の黒を追加しました。
- カスタムカラーテーマで文字色と背景色のコントラストが不足する場合、白または黒の読みやすい文字色へ自動補正するようにしました。アクセント色上のボタン文字も別に判定します。

### ギャラリー
- フォルダ/ギャラリー画面の長押し範囲選択を、グリッド全体の root 座標で扱う方式に見直し、画面上下端での自動スクロール、範囲追従、選択モード終了後の誤選択を安定化しました。
- 長押し範囲選択のスクロール速度を、端への近さ・画面外への距離・保持時間で段階的に変わるように調整し、原因調査用の `GALLERY_SELECTION_TRACE` ログを追加しました。
- 選択中の復帰スクロールで元の位置へ戻ってしまうケースを抑制し、`GalleryGridView` の nullable 型エラーを修正しました。
- ギャラリーのサムネイル読み込みを、列数と画面幅に合わせたデコードサイズと近傍優先の軽量先読みへ調整しました。
- ライトモードでも3点ボタンや選択中メニューが見えるよう、トップバー/メニュー/スクロールバー/バッジ周辺の色をテーマ色に合わせました。

### ビュワー
- メディア画像とブックビューアの長押し拡大鏡を、長押し検出・追従・復元を同じジェスチャ処理で扱うように修正し、指を動かしても拡大位置が途切れにくいようにしました。
- メディア/動画/ブックのボトムバー設定を整理し、動画内ギャラリーの上スワイプ時にプレビュー動画やトップバーがずれたり、画面下部に帯が出たりしにくいよう調整しました。
- ブックビューアでしおりの追加、切り替え、解除、しおり画面からの解除に対応し、シークバー下部の回転/しおりボタンをトップバー3点メニューへ移動しました。
- Xperia など一部端末の動画再生で音が出ないケースに備えて、動画再生側の音声属性とプレイヤー制御を見直しました。

### 設定・バックアップ
- 設定ファイル読み込み時に `Integer` と `Float` が混在してもクラッシュしないよう、数値設定の読み込みを堅牢化しました。
- 設定/お気に入りの読み込みは既存データを消さずに取り込み、タイトルなどが重複する項目は上書きしない方針へ変更しました。
- 設定画面で長押し選択の間隔を変更できるようにし、単位表示が `%%` になっていた箇所を `%` に修正しました。
- AI分析速度をバランス/高速/精度優先から選べるようにし、通常温度時の待機時間とタグ保存処理を軽量化しました。
- 普通AI/高性能AIの選択に対応し、普通AIで解析済みの画像を高性能AIで再解析できるようにしました。
- スクリーン割り当てとタッチ様式割り当ての分割数と配置を揃え、ボトムバー/タッチ様式のカタログは長押し中に指へ追従し、ボタンに重なった瞬間に白枠フィードバックを出すようにしました。
- ボトムバー配置にデフォルト配置を用意し、再生/停止アクションをボトムバー設定から外しました。

### お気に入り・ファイル操作
- お気に入りサイト/お気に入りクリエイターで、検索ページそのものをURLに誤登録しないようにし、URLがある候補の長押し/取り込み操作で登録先へ代入できるようにしました。
- お気に入りサイト/クリエイターリンクのURL正規化、重複抑制、クリエイター空表示、ユーザーサイト候補の削除に対応しました。
- ファイルのフォルダ移動、ゴミ箱からの復元、動画ファイルを含む復元処理の扱いを見直しました。

### バージョン
- アプリバージョンを 0.5.0 から 0.6.0 に更新しました。
