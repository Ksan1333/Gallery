# 更新履歴 (CHANGELOG)

## [1.1.0] - 2026-07-21

### ギャラリー操作・スクロール
- サイドバーの右スワイプとギャラリーの上下スクロールが競合しないよう、上下スクロールを優先するジェスチャー判定へ見直しました。右端付近でもギャラリーを上下にスクロールできます。
- スクロールやピンチズーム、右スワイプが画像タップとして扱われにくいようにし、意図しない画像表示を抑制しました。
- 長押し範囲選択をタイル単位の長押し後ドラッグとして再実装し、ドラッグ中は選択モード用UIの切替を保留することで、選択範囲の更新が中断されにくいようにしました。原因確認用に `GALLERY_SELECTION_TRACE` の詳細ログも追加しました。
- スクロールバーの追従、上下端の到達表示、ギャラリー下端との整合性を改善しました。年月ラベルの表示幅と操作領域も見直しています。

### タグ・一括編集
- 一括編集画面をカード型のレイアウトへ刷新し、タイトルに選択件数を表示するようにしました。画面全体をスクロールでき、キーボード表示時にも編集領域を使いやすくしています。
- 一括編集のタグ入力は1つの入力欄へ集約しました。入力内容で下の既存タグ一覧を絞り込み、未入力時は使用数上位20件のみを表示します。
- 一括編集のタグ候補ポップアップを廃止し、キーボード操作を遮る表示や画面下部の帯が出ないようにしました。
- レコメンド画面から直接タグを追加できるようにし、タグ入力では既存タグの候補検索を利用できるようにしました。ひらがな・カタカナの表記ゆれも検索対象として扱います。
- レコメンドのタグから条件検索へ移動した後、戻る操作で条件なしのギャラリーへ戻るようにしました。

### レコメンド・メディアビューア
- レコメンド表示サイズを「最大（画面全体）」と「最小（従来の42%）」から選べるようにしました。初期設定は最大で、選択中の項目は青い枠で確認できます。
- レコメンドを最大表示している間は、OSナビゲーションバーを表示するようにしました。
- 動画ビューアの3点メニューおよび編集系ボトムバーからGIFコマ送りを削除しました。
- 動画ビューアで一時的に回転した後、ビューアを閉じた際に画面回転状態が残らないよう修正しました。

### バージョン
- アプリバージョンを `1.0.1` から `1.1.0` へ更新しました。

## [0.8.0] - 2026-07-20

### 動画ビューア
- シークプレビュー機能を大幅に強化しました。
  - **ハイブリッド・キャッシュ**: シークバー操作時は動画全体をカバーする「25分割・高速キャッシュ」、画面左右スワイプ時は現在地周辺を精細に映す「10分割・高画質キャッシュ（720p）」を自動で使い分けるようにしました。
  - **ローリング・キャッシュ**: 画面スワイプを開始した瞬間、前後30秒（合計1分間）の範囲を高画質で即座に抽出し、遅延のない鮮明なフレーム移動を可能にしました。
- 動画フォルダ画面のトップバーをメイン画面と共通のデザインに統一し、設定アイコンやタイトル表示の整合性を改善しました。

### ブックビューア
- Kindleスタイルの高度なシークナビゲーションを実装しました。
  - **静止メイン画面**: シークバー操作中、背後のページを固定したまま、中央のプレビューカードで内容を確認できるようにしました。現在地を見失うことなく高速にページを探せます。
  - **全ページ・プリキャッシュ**: 本を開くと同時に全ページの軽量サムネイルを裏側で生成し、シーク中の読み込み待ちを解消しました。
  - **シークアンカー（記憶点）**: シークを開始した地点に「アンカー」を自動配置し、磁石（スナップ）効果で正確に戻れるようにしました。スナップ範囲を1%に微調整し、1ページ単位の移動と吸着のしやすさを両立しました。
- 設定画面にアンカーの有効化スイッチと、最大記憶数（1〜5個）のカスタマイズ項目を追加しました。

### 設定・UI
- 設定画面の構成を再編し、ナビゲーション順序を「全般 -> テーマ -> メディア -> 動画 -> ブック」に整理しました。
- 設定画面の各ラベルに `galleryTypography` を適用し、視認性を高めるためフォントサイズを拡大しました。
- キャッシュ管理を独立したセクションに分離し、「キャッシュの消去」機能の動作（サムネイルやバッファの削除、データベースの保持）に関する説明を詳細化しました。

## [0.7.0] - 2026-07-12

### ギャラリー・フォルダ操作
- フォルダ画面のデフォルト表示を3列に変更しました。
- フォルダを長押しドラッグして自由に並び替えられるようにしました。ドラッグ直後に位置が戻ってしまう不具合も修正しました。
- フォルダ内のメディアをすべて移動または削除して空になった際、自動的にフォルダ一覧画面へ戻るようにしました。
- フォルダ移動時の移動先選択画面でも、フォルダグループや並び替え順序が正しく反映されるように改善しました。
- フォルダ画面で長押しメニューによる操作と、ドラッグによる並び替えを両立できるよう調整しました。競合していた3点ボタンは削除し、UIをクリーンに保ちました。
- フォルダのグループ化機能を改善し、既存のグループに新しいフォルダを直接追加できるようになりました（1つのグループと他の項目を選択してグループ化した場合）。
- グループ内のタイトル横の編集アイコンから、グループ名を変更できるようになりました。
- グループ解除およびグループ内フォルダの個別解除時に確認ダイアログを表示するようにし、誤操作を防止しました。
- グループ操作（作成・追加・解除・名前変更）および並び替えの結果が画面に即座に反映されるように改善しました。
- 類似画像グループ化の判定基準（一致率）を設定画面から 1%〜100% の範囲で調整できるようにしました。

- 2列表示を段違いグリッドへ変更し、横並び画像の高さを揃えず、各画像の縦横比を保ちながら空いた領域を次の画像で埋めるようにしました。
- 4列から7列、7列から28列へ切り替える際は、負荷の高いアイテム移動・サイズ変更アニメーションを行わないようにしました。
- スクロールバーは短いタップでは移動せず、長押し後だけドラッグへ切り替わるようにしました。長押し判定と追従間隔を調整し、`GALLERY_SCROLLBAR_TRACE` で操作を追跡できます。
- ギャラリーのサムネイルデコードサイズ、近傍先読み、スクロール中のロード抑制を見直し、表示開始と高速スクロールの応答性を改善しました。
- 検索画面に過去の検索条件を復元できる履歴を追加し、保存件数を設定から1〜10件で変更できるようにしました。

#### メディア・動画・ブックビューア
- メディアビューワーでレコメンドを表示したままでも、画像領域の左右スワイプで前後のメディアへ移動できるようにしました。ビューワー表示中はサイドバーを開かないようにしました。

- 動画ビューワーの3点メニューは操作後に自動で閉じないようにし、トップバーの設定ボタンとコマ送りボタンを整理しました。
- ブックビューワーの全分割タイプにタッチ様式の初期割り当てを用意し、しおり追加・切り替え・解除とトップバーからの回転・スライドショー操作を整理しました。
- 画像平滑化設定の各方式について、処理内容と向いている画像を箇条書きで確認できる説明を追加しました。

#### AI分析・ダウンロード・表示
- AI分析実行ダイアログから普通AIと高性能AIを選択できるようにし、普通AIで解析済みの画像を高性能AIで再解析できるようにしました。
